# Brainstorming de Diseño: Centro de Psicología Petrer & Elda

## Contexto
Diseñar una web para un centro de psicología especializado en terapias individuales, de pareja y familiar, optimizada para SEO local en Petrer y Elda. La interfaz debe transmitir profesionalidad, calidez, empatía y confianza.

---

## Idea 1: Minimalismo Terapéutico (Probabilidad: 0.08)

### Design Movement
**Bauhaus moderno + Psicología del color minimalista**. Inspirado en espacios terapéuticos reales: consultorios con paredes neutras, luz natural y elementos que respiran.

### Core Principles
1. **Espacio negativo como herramienta**: El silencio visual comunica calma y apertura mental.
2. **Tipografía como comunicación emocional**: Fuentes sans-serif limpias para profesionalismo, serif para calidez en títulos.
3. **Paleta monocromática con acentos**: Blanco/gris neutro + un único color de acento (verde salvia o azul profundo).
4. **Funcionalidad extrema**: Cada elemento tiene propósito; nada es decorativo.

### Color Philosophy
- **Fondo**: Blanco puro (confianza, limpieza) o gris muy claro (calidez sin frialdad).
- **Acento primario**: Verde salvia (crecimiento, equilibrio, naturaleza) o azul profundo (calma, profesionalismo).
- **Textos**: Gris oscuro (no puro negro, menos agresivo).
- **Intención emocional**: Transmitir que el espacio es seguro, ordenado y libre de distracciones.

### Layout Paradigm
- **Asimetría controlada**: Texto a la izquierda, espacios en blanco a la derecha.
- **Secciones verticales largas**: Scroll lento y meditativo.
- **Márgenes amplios**: Respeto por el espacio personal del usuario.
- **Grilla invisible**: Alineación perfecta sin líneas visibles.

### Signature Elements
1. **Líneas horizontales sutiles**: Separadores minimalistas que dividen secciones sin ser invasivos.
2. **Bloques de color sólido**: Fondos de color acento en áreas clave (CTAs, testimonios).
3. **Tipografía escalada**: Tamaños muy contrastados (títulos enormes, cuerpo pequeño).

### Interaction Philosophy
- **Transiciones suaves**: Hover effects que revelan información gradualmente.
- **Micro-interacciones silenciosas**: Cambios sutiles que no distraen.
- **Botones minimalistas**: Bordes finos, sin relleno, que se activan al pasar el ratón.

### Animation
- **Entrada de elementos**: Fade-in lento (300-500ms) al hacer scroll.
- **Hover en botones**: Cambio de color suave + ligero movimiento hacia arriba.
- **Formularios**: Campos que se expanden suavemente al enfocarse.
- **Scroll parallax suave**: Fondos que se mueven ligeramente más lento que el contenido.

### Typography System
- **Display**: Playfair Display (serif elegante) para títulos principales.
- **Body**: Inter (sans-serif limpio) para contenido.
- **Jerarquía**: H1 (56px), H2 (36px), H3 (24px), Body (16px), Small (14px).

## Idea 2: Calidez Contemporánea (Probabilidad: 0.07)

### Design Movement
**Diseño humanista + Ilustración moderna**. Inspirado en estudios de diseño que priorizan la conexión emocional: colores cálidos, formas orgánicas, ilustraciones personalizadas.

### Core Principles
1. **Humanidad en cada píxel**: Formas redondeadas, ilustraciones dibujadas a mano, colores naturales.
2. **Narrativa visual**: Cada sección cuenta una historia sobre el viaje terapéutico.
3. **Accesibilidad emocional**: Lenguaje visual que se siente cercano, no corporativo.
4. **Comunidad y pertenencia**: Énfasis en testimonios, equipo visible, espacios compartidos.

### Color Philosophy
- **Primario**: Terracota cálido (seguridad, tierra, naturaleza) o coral suave (energía, calidez).
- **Secundario**: Crema/beige (comodidad, hogar).
- **Acentos**: Verde menta (crecimiento, frescura) o azul cielo (calma, esperanza).
- **Intención emocional**: Sentirse como entrar en un hogar acogedor, no en una oficina.

### Layout Paradigm
- **Flujo orgánico**: Secciones con formas ondulantes, no rectángulos perfectos.
- **Asimetría natural**: Imágenes y texto que se superponen de forma orgánica.
- **Secciones alternadas**: Imagen izquierda/texto derecha, luego invertido.
- **Uso de diagonales**: Divisores en ángulo que crean movimiento.

### Signature Elements
1. **Ilustraciones personalizadas**: Dibujos que representan emociones, terapia, crecimiento.
2. **Formas orgánicas**: Círculos, óvalos, formas libres (no rectángulos).
3. **Texturas sutiles**: Patrones de fondo que evocan naturaleza (líneas, puntos, tejidos).

### Interaction Philosophy
- **Animaciones fluidas**: Elementos que flotan, rebotan suavemente.
- **Respuesta táctil**: Botones que se sienten "presionables", con sombra que cambia.
- **Revelación progresiva**: Información que se muestra al interactuar.

### Animation
- **Entrada de elementos**: Slide-in desde los lados + fade-in simultáneo.
- **Hover en botones**: Escala ligera + cambio de color + sombra expandida.
- **Ilustraciones**: Movimiento sutil (respiración, parpadeo).
- **Scroll**: Elementos que se desplazan a diferentes velocidades (parallax orgánico).

### Typography System
- **Display**: Poppins Bold (sans-serif amigable) para títulos.
- **Body**: Lato (sans-serif cálido) para contenido.
- **Jerarquía**: H1 (48px), H2 (32px), H3 (22px), Body (16px), Small (14px).

## Idea 3: Profesionalismo Moderno (Probabilidad: 0.06)

### Design Movement
**Diseño corporativo elevado + Tendencias 2025**. Inspirado en startups de salud mental y clínicas premium: gradientes sutiles, tipografía audaz, componentes geométricos.

### Core Principles
1. **Sofisticación moderna**: Gradientes suaves, sombras estratégicas, profundidad visual.
2. **Claridad extrema**: Información bien jerarquizada, fácil de escanear.
3. **Confianza a través del diseño**: Profesionalismo que transmite experiencia y credibilidad.
4. **Innovación visual**: Elementos que se sienten actuales, no anticuados.

### Color Philosophy
- **Primario**: Azul profundo (profesionalismo, confianza, medicina).
- **Secundario**: Púrpura suave (creatividad, transformación).
- **Acentos**: Dorado/amarillo cálido (optimismo, éxito).
- **Gradientes**: Transiciones suaves entre colores primarios y secundarios.
- **Intención emocional**: "Soy profesional, moderno y conozco mi oficio."

### Layout Paradigm
- **Grilla moderna**: Alineación perfecta con espacios generosos.
- **Secciones de dos columnas**: Texto + imagen/gráfico balanceados.
- **Tarjetas flotantes**: Elementos que parecen flotar sobre el fondo.
- **Uso de fondos gradiente**: Sutiles, no agresivos.

### Signature Elements
1. **Gradientes direccionales**: Fondos que transicionan de un color a otro.
2. **Tarjetas con sombra elevada**: Profundidad visual clara.
3. **Iconografía moderna**: Iconos geométricos, no ilustraciones.

### Interaction Philosophy
- **Transiciones elegantes**: Cambios que se sienten premium.
- **Feedback visual claro**: Usuarios saben exactamente dónde están.
- **Botones con movimiento**: Animaciones que se sienten responsivas.

### Animation
- **Entrada de elementos**: Scale-up + fade-in (elementos que "aparecen").
- **Hover en botones**: Cambio de gradiente + movimiento sutil.
- **Tarjetas**: Elevación al pasar el ratón (sombra aumenta).
- **Scroll**: Fade-in de elementos + ligero cambio de opacidad.

### Typography System
- **Display**: Montserrat Bold (sans-serif moderno) para títulos.
- **Body**: Open Sans (sans-serif legible) para contenido.
- **Jerarquía**: H1 (52px), H2 (34px), H3 (24px), Body (16px), Small (14px).

---

## Decisión Final

Después de evaluar las tres opciones, **selecciono: Idea 2 - Calidez Contemporánea**.

### Razones:
1. **Diferenciación**: Mientras que los competidores usan diseños corporativos fríos, este enfoque humanista nos destaca.
2. **SEO + Experiencia**: Las ilustraciones personalizadas mejoran el engagement y tiempo en página (factores de SEO).
3. **Conversión**: La calidez visual reduce la fricción psicológica para contactar; la gente se siente más cómoda.
4. **Escalabilidad**: Las formas orgánicas y paleta cálida funcionan bien en móvil y desktop.
5. **Identidad**: Transmite que "aquí te entendemos, aquí eres bienvenido" — exactamente lo que busca un paciente potencial.

### Implementación:
- **Colores**: Terracota cálido (#C97A5C) + Crema (#F5E6D3) + Verde menta (#7FB3A0) + Azul cielo (#A8D5E2).
- **Tipografía**: Poppins Bold + Lato.
- **Ilustraciones**: Generaré 3-5 ilustraciones personalizadas para hero, servicios y equipo.
- **Animaciones**: Fluidas, orgánicas, nunca jarring.
